Este repositório é um portfólio pessoal e projeto open-source. Todos os visitantes e colaboradores são bem-vindos, independente de experiência, origem ou background.

✅ Respeitar outros jogadores no leaderboard e issues  
✅ Reportar bugs de forma construtiva
